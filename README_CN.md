# Vehicula - 理想L9汽车配置器

基于 Qt Quick 3D 的理想L9（Li Auto L9）电动汽车配置器。

## 概述

Vehicula 是一款交互式3D汽车配置器，允许用户探索和自定义理想L9汽车模型。基于 Qt Quick 3D 技术构建，提供逼真的3D渲染体验和多种交互功能。

## 功能特性

### 3D车辆可视化
- 理想L9高精度3D模型
- HDR环境光照实时渲染
- 可调节相机角度和位置

### 交互控制
- **车门控制**：平滑动画开关4个车门
- **后备箱控制**：开关后备箱盖
- **车轮转向**：可调前轮转向角度
- **车漆颜色**：实时颜色自定义
- **车灯控制**：开关前大灯

### 相机系统
- **车身视角**：默认外观视角
- **内饰视角**：驾驶员座位透视
- **环绕动画**：360°自动环绕车辆

### 空调系统
- 左右独立温度控制（16℃ - 32℃）
- 5档风速调节
- 空调电源开关
- 3D粒子风效可视化

### 灯光控制
- 可调方向光亮度
- 环境探针曝光控制
- 内饰显示屏发光因子

## 系统要求

- Qt 6.9.3 或更高版本
- Qt Quick 3D 模块
- MinGW 64位编译器（Windows）
- Windows 10/11

## 构建说明

### 使用 Qt Creator
1. 在 Qt Creator 中打开 `Vehicula.pro`
2. 选择 Qt 6.9.3 MinGW 64位工具包
3. 点击 构建 → 构建项目 "Vehicula"

### 使用命令行
```bash
mkdir build
cd build
qmake ..\Vehicula.pro
mingw32-make
```

## 项目结构

```
Vehicula/
├── main.cpp                    # 应用程序入口
├── main.qml                    # 主界面和3D视口
├── QuickButton.qml             # 自定义按钮组件
├── QuickSliderBar.qml          # 风速滑块组件
├── Vehicula.pro                # Qt项目文件
├── resources.qrc               # Qt资源文件
├── LiXiang_L9/
│   ├── LiXiang_L9.qml         # 3D汽车模型组件
│   ├── meshes/                 # 3D网格文件
│   └── source scene/          # 原始FBX源文件
├── maps/                       # 纹理贴图
├── textures/                   # HDR环境贴图
└── images/                     # UI图标和粒子纹理
```

## 操作说明

| 控制项 | 说明 |
|--------|------|
| 左/右箭头 | 调节温度 |
| 风扇滑块 | 调节风速（0-5档） |
| 电源按钮 | 开关空调 |
| 内饰按钮 | 切换到内饰视角 |
| 车身按钮 | 返回车身视角 |
| 环绕按钮 | 开始/停止环绕动画 |
| 车门按钮 | 开关车门 |
| 后备箱按钮 | 开关后备箱 |
| 车灯按钮 | 开关前大灯 |
